. Why EC2 specifically, and what else does AWS offer?
EC2 (Elastic Compute Cloud) = a virtual server you rent by the hour/month. You chose it because you needed a full Linux machine capable of running Docker — something with a real CPU, OS, and root access, since Docker needs to manage containers, networking, and processes at the OS level.
Other AWS services worth knowing about, especially since you're deploying apps:
